# Smart Crowd Panic Detection System

An industrial safety and surveillance system for real-time crowd monitoring, anomaly detection, and risk assessment in public or high-traffic environments.

## Features

- Human detection and tracking
- Crowd density analysis
- Motion pattern analysis
- Panic and anomaly detection
- Heatmap generation and risk scoring
- Real-time video processing and visualization

## Use Cases

- Public safety monitoring
- Disaster prevention
- Surveillance systems
- Crowd management in events and industrial zones
- Emergency response support

## Project Structure

- app.py - Main application for detection, tracking, visualization, and output generation
- yolov8n.pt - Pretrained YOLOv8 model weights
- output_crowd_panic_ai.mp4 - Example output video generated by the system

## Requirements

Install the required Python packages:

```bash
pip install ultralytics opencv-python numpy torch torchvision scipy matplotlib Pillow supervision
```

If you are using the Anaconda environment and encounter dependency issues with NumPy/SciPy/Matplotlib, use a compatible stack such as:

```bash
pip install --upgrade --force-reinstall "numpy<2" "scipy==1.13.1" "matplotlib==3.8.4"
```

## Running the Application

From the project folder, run:

```bash
python app.py
```

You can also pass a video file, webcam index, or RTSP stream as input:

```bash
python app.py 0
python app.py path/to/video.mp4
```

## Controls

- Q: Quit the application
- P: Pause or resume processing
- S: Save a screenshot

## Output

The system generates:

- A processed video output saved as output_crowd_panic_ai.mp4
- Live visualization with crowd map, panic score, and HUD overlays

## Notes

This system uses YOLOv8 for person detection and combines tracking, optical flow, density analysis, and panic scoring to identify abnormal crowd behavior in real time.
